------------------------------------------------------------
  操作步骤 / 操作手順
------------------------------------------------------------

中文：
1. 准备上面清单中的图片，按表格中的「文件名」重命名
2. 将所有图片文件复制到这个 images/ 文件夹
3. 在终端执行以下命令上传到网站：
   cd fuluckpet-website
   git add images/
   git commit -m "Add website images"
   git push

日本語：
1. 上記リストの画像を準備し、「ファイル名」の通りにリネーム
2. すべての画像ファイルをこの images/ フォルダにコピー
3. ターミナルで以下のコマンドを実行してサイトにアップロード：
   cd fuluckpet-website
   git add images/
   git commit -m "Add website images"
   git push
