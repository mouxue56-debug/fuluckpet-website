# 员工操作教程 — 网站后台管理

> 本教程面向福楽キャッテリー的工作人员。
> 不需要任何技术知识，按步骤操作即可。

---

## 一、管理面板操作

### 登录管理面板

1. 在浏览器中打开：`https://fuluckpet.com/admin/`
2. 输入密码（找管理员获取）
3. 登录成功后会看到仪表盘

### 添加新小猫

1. 点击左侧菜单 **子猫管理**
2. 点击右上角 **＋ 新規追加**
3. 填写以下信息：
   - **Breeder ID**（如 `2602-00625`）
   - **品种**（如 サイベリアン）
   - **性别**（♂ 或 ♀）
   - **毛色**（如 ブルーポイント）
   - **生日**（如 2025-12）
   - **价格**（数字，如 250000）
   - **状态**（販売中 / 商談中 / ご家族決定）
   - **パパ猫 / ママ猫 名前**（父母猫名字）
4. 点击 **保存**
5. 如需上传照片，点击 **写真** 按钮上传猫咪照片（建议 3-8 张）
6. 完成后点击右上角 **📤 発行する**（发布更新）

### 修改小猫信息

1. 在子猫管理列表中找到要修改的猫咪
2. 点击该猫咪行的 **编辑** 按钮
3. 修改信息后点击 **保存**
4. 点击 **📤 発行する** 发布更新

### 小猫已售出

1. 找到猫咪 → 点击编辑
2. 把状态改为 **ご家族決定**
3. 保存 → 点击 **📤 発行する**

### 关于「📤 発行する」按钮

- 每次修改完数据后，建议点一下右上角的 **📤 発行する** 按钮
- 这会更新搜索引擎（Google）能看到的静态页面
- **不点也不影响网站显示**——网站会实时读取最新数据
- 点击后约 **2分钟** 静态页面会自动更新
- 每天凌晨 3 点也会自动更新一次（不需要手动操作）

## 二、通过 Google Drive 上传照片

### 准备工作（只需做一次）

确保你的 Google 账号已被添加到共享文件夹 `fuluckpet-photos` 的访问权限中。
如果还没有权限，请联系管理员将你的 Google 邮箱添加为"编辑者"。

### 文件夹结构说明

```
fuluckpet-photos/
├── kittens/              ← 在售子猫照片
│   ├── 2602-00625/       ← 每只子猫一个文件夹（用 breeder ID 命名）
│   ├── 2511-02287/
│   └── ...
├── parents/              ← 种猫（爸爸妈妈）照片
│   ├── しろくん/          ← 用猫名命名
│   ├── りんちゃん/
│   └── ...
└── gallery/              ← 已毕业猫咪（客户家的猫）照片
    ├── 01/
    ├── 02/
    └── ...
```

### 添加新子猫照片

1. 打开 Google Drive：https://drive.google.com
2. 进入 `fuluckpet-photos` → `kittens` 文件夹
3. 点击 **"+ 新建"** → **"文件夹"**
4. 文件夹名称填写子猫的 **Breeder ID**（例如：`2602-00625`）
5. 打开刚创建的文件夹
6. 将猫咪照片拖入文件夹（或点击"+ 新建" → "上传文件"）
7. 完成！网站会在 **30 分钟内** 自动显示新照片

### 添加种猫 / 毕业猫照片

- 种猫：进入 `parents` → 新建文件夹（猫名命名）→ 拖入照片
- 毕业猫：进入 `gallery` → 新建文件夹（编号命名）→ 拖入照片

### 更新已有猫咪的照片

1. 找到对应的文件夹（如 `kittens/2602-00625`）
2. 直接拖入新照片即可
3. 如果要删除某张照片，右键 → 移到垃圾桶
4. 照片会按文件名排序显示，第一张会作为封面图

## 三、照片要求

| 项目 | 建议 |
|------|------|
| **格式** | JPG 或 PNG（推荐 JPG，文件更小） |
| **大小** | 每张照片建议 1-2MB 以内（超过 2MB 系统会自动压缩） |
| **尺寸** | 建议 1200x900px 以上（太小会模糊） |
| **数量** | 每只猫建议 3-8 张照片 |
| **命名** | 建议用 `01.jpg`、`02.jpg` 来控制顺序 |

### 拍照小贴士

- 光线充足的环境拍摄效果最好
- 横拍（landscape）适合展示猫咪全身
- 竖拍（portrait）适合展示猫咪特写
- 第一张照片会作为列表封面，选一张最可爱的！

## 四、常见问题

### Q: 修改了数据但网站没有变化？
A: 网站数据是实时加载的，刷新页面即可看到变化。如果仍有问题，尝试清除浏览器缓存或使用隐私模式查看。

### Q: 点了「発行する」有什么效果？
A: 它会更新 Google 搜索能看到的静态页面。对于普通访客没有影响（他们看到的是实时数据）。约 2 分钟后生效。

### Q: 我上传了照片但网站没有显示？
A: 请等待最多 30 分钟（系统缓存时间）。如果急需更新，去管理面板点击"缓存刷新"。

### Q: 照片顺序不对怎么办？
A: 照片按文件名字母顺序排列。建议用 `01.jpg`、`02.jpg`... 来控制顺序。

### Q: 可以上传视频吗？
A: 目前不支持通过 Drive 同步视频。如需上传视频请联系管理员。

### Q: 删除了文件夹会怎样？
A: 对应猫咪的照片会从网站上消失（缓存过期后）。请谨慎操作！
