# FABLE_VERDICTS — 裁决 / 砍单 / 开放问题

> 本地文档，勿提交公开仓库。施工顺序：**U3 & D3（信任+数据，天级）→ D1（管线）→ D2（骑 D1 上）→ D4（可并行）→ UI 波次（U1→U4→U2）**。

## 裁决表
| 编号 | 裁决 | 理由 | 验收 |
|---|---|---|---|
| D1 | **准 B**（minify 挂进生成器） | 静态站要字节减法不是构建革命；C 复杂度买不到收益 | 产物 min + 内容哈希 ?v=；源码可读；首页 CSS+JS（br 后）≤60KB；零视觉回归 |
| D2 | **准 C 限定范围 + B 兜底** | 客户端 JS 翻译对 SEO 是二等公民，转化页必须静态三语；×3 只对生成器模板页成立，长尾强上是浪费 | 子猫列表/详情+cornerstone 静态 /en//zh/+hreflang 进 sitemap；其余页 i18n 按页拆分 ≤15KB；98KB 单文件不再出现 |
| D3 | **准 B+C，C 升级为「构建失败」而非告警** | 告警会被无视；重复 ID 是数据损坏，录入不可能+构建致命，静默覆盖零容忍 | admin 撞号被拒并报错；生成器遇重复 exit 1；sitemap 去重降为第三道保险；owner 清脏数据后每只有唯一 URL |
| D4 | **准 B**（R2+边缘缓存） | 把 Drive 移出热路径、保留 owner「丢图即生效」工作流；C 让图片与部署耦合还撑爆 repo | 缓存键=Drive fileId+版本，Cache-Control immutable 长 TTL；同图二次请求 CF HIT；详情页 LCP 改前后各测留档 |
| U1 | **改选：原生排版组件替换截图** | 截图重/不可选中/翻译/响应式且像贴图；逐字转录即无编造风险 | 两 PNG(~500KB)删除；文字可选中可翻译；与截图逐字一致+出典链接 |
| U2 | **降级：diary 撤出主导航，URL 保留+设计空态** | 空着的旗舰位比没有更伤信任 | 主导航+首页 teaser 移除；/diary/ 按 §4.6 空态；owner 交付≥3 篇后恢复入口 |
| U3 | **立即修·全单最高优先级** | 页面/schema 与实价不符=信任+景品表示法风险，高于一切性能项 | 价格区间构建期从真实在售数据算(min/max)，页面与 schema 同源；grep 全站无手写价格残留 |
| U4 | **准做，按 §4 亲笔执行** | 记忆点落在真实差异化(低致敏+全国1位)，不靠新配色 | §4.4 三件签名元素落地；1 秒瞥视能与通用猫舍模板区分 |
| U5 | **否决独立专项** | 收益已被 D1(minify)+D2(拆分/静态化)吃掉 | 以 D1/D2 验收为准 |

## 砍单
- 砍 D1-C（Vite/esbuild 全构建）：复杂度换不来静态站需要的东西。
- 砍本轮 CSS 死代码清除（Purge）：误删风险>收益，minify 先行，Purge 推迟到有视觉回归测试后。
- 砍「全站页面×3 静态化」：只做生成器模板页+cornerstone，长尾走 i18n 按页拆分。
- 砍 D4-C（图片拉本地）：repo 体积与部署耦合双输。
- 砍 U5 独立专项：并入 D1/D2。
- 砍 diary 主导航位：推迟到 ≥3 篇真实内容。
- 砍 U1 引入 JS 组件/轮播库：纯 HTML/CSS 卡片即可。
- 推迟 Worker 端图片 WebP/AVIF 现转：D4-B 落地实测 LCP 后再裁。
