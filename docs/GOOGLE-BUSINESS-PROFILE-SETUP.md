## 二、操作流程（首次设立）

### Step 1 · 创建账号（如果没有）

1. 浏览器打开 https://www.google.com/business/
2. 用现有 Google 账号登录（建议用业务专用账号，不是个人 Gmail）
3. 点 "Manage now"

### Step 2 · 输入业务信息

```
Business name:    福楽キャッテリー
                  (假名 / 罗马字: Fukuraku Cattery 可加在「Add additional name」)

Category (主):    Pet breeder
Categories (副):  Cat breeder
                  Animal protection organization (可选)

Service area:     入力する
Address:          大阪府大阪市城東区東中浜 [街番]
                  (准确地址 — 如果想保护隐私可以选 "Hide address" 后只显示
                   服务范围；但建议显示完整地址 — 真实性是 Google 排名因素)

Phone:            080-5416-7843
                  (这是你给我配的号码，会显示给客户。你可以在 GBP 后台
                   设置 "Call forwarding" 转到 LINE / 留言信箱避免直接接
                   日语电话)

Website:          https://fuluckpet.com
```

### Step 3 · 验证身份

Google 通过 3 种方式之一验证你是真实业务主：

1. **明信片邮寄** — Google 寄一张含验证码的明信片到你地址，14 天内收到。最常用。
2. **电话** — 自动机器人电话给你 080-5416-7843 念验证码。
3. **视频** — 录像证明你在该地址（手机录店铺外观 + 内部 + 你本人 + 业务证明文件如动物取扱业許可証）。**最快**，1-3 天审。

> 推荐 **方法 3 视频** — 你的店铺装修完后录一段 1-2 分钟视频包含：
> - 店铺外观（看清门牌 + 街景）
> - 内部业务证据（猫笼 / 第一种動物取扱業 220012A 牌子）
> - 你本人（无需说话，挥手即可）
> 上传后 Google 1-3 工作日审核。

### Step 4 · 完善 Profile（关键 — 这部分决定排名）

完成验证后立刻填以下内容（按重要性）：

#### 4.1 Description（业务介绍）— 最多 750 字
```
福楽キャッテリーは、大阪府大阪市城東区にあるサイベリアン専門のブリーダーです。
代表 羅方遠（ラホウエン）が運営する福楽株式会社として、第一種動物取扱業
220012A（〜2027年4月）+ 240051A（〜2029年7月）の登録のもと、これまでに
200頭以上の子猫をご家族にお届けしてきました。

2025年には「みんなの子猫ブリーダー」で
- 全国上半期サイベリアンブリーダーお客様評価 第1位
- 2025年下半期 全国第2位 / 大阪府第1位
を受賞いたしました。

サイベリアンは、Fel d1タンパク質が比較的少ないとされる低アレルゲンの天然猫種
として国内外で注目を集めています。当キャッテリーでは、わんちゃん・うさぎ・
フェレットと共に暮らす環境で自然な社会化を行い、毎日の清掃と空気清浄システムで
衛生管理を徹底しています。

ブリティッシュショートヘア、ブリティッシュロングヘア、ラグドールも取り扱います。
完全予約制の見学（対面 / LINEビデオ通話）から、お迎え後の生涯LINEサポートまで、
末永くお付き合いできる関係を目指しています。

価格帯：¥160,000〜¥290,000（税込）
LINEでのご相談を歓迎します。
```

#### 4.2 营业时间
- 完全予約制 → "By appointment" 选项
- 或者设个宽口范围如 10:00-18:00 + 注释 "見学は事前 LINE 予約必須"

#### 4.3 服务区域
设 「Service area」覆盖：
- 大阪府全域
- 兵庫県
- 京都府
- 奈良県
- 和歌山県
- 滋賀県

（与 LocalBusiness schema 的 areaServed 一致）

#### 4.4 上传照片（≥10 张）

- ✅ 店铺外观 1-2 张
- ✅ 内部环境（清洁 / 空气清浄机 / 笼舍）3-4 张
- ✅ 在售子猫 5-10 张（高质量 lifestyle 照）
- ✅ 親猫展示 2-3 张
- ✅ Logo 标志（已有）
- ✅ 卒業猫与新主合照（要客户授权）2-3 张

照片越多 + 越新 = 排名越高。

#### 4.5 服务列表（Services）

逐个添加：
- サイベリアン子猫販売
- ブリティッシュショートヘア子猫販売
- ブリティッシュロングヘア子猫販売
- ラグドール子猫販売
- 見学予約
- 全国空輸対応
- アフターサポート（LINE）
- 親猫展示

每个服务可以加 description + 价格 + 图片。

#### 4.6 属性（Attributes）

勾选：
- ✅ 完全予約制
- ✅ 駐車場あり（如有）
- ✅ オンライン相談あり（LINE）
- ✅ クレジットカード非対応 / 銀行振込のみ
- ✅ Pet adoption available

## 四、首批评价获取

GBP 没评价的话排名很低。**第一周内争取 5-10 条评价**：

发 LINE 给最近 5-10 位忠实老客户：
```
〇〇さま、いつもお世話になっております🐾
このたび福楽キャッテリーが Google ビジネスプロフィールに登録されました。
お差し支えなければ、Googleでのレビュー（口コミ）にてお迎え後のご感想を
シェアいただけますと、これから猫のお迎えを検討されている方のご参考になり
本当に嬉しいです。

→ Google レビューリンク：[GBP_URL]/review
```

每个评价回复（"〇〇さま、〇〇ちゃんのご様子をお聞かせいただきありがとうございます🐾"）
— 回复率高 = Google 评估你的活跃度好。

## 八、问题排查

| 症状 | 解决 |
|---|---|
| 验证明信片 14 天没到 | 后台重新申请 + 检查地址 |
| 视频审核被拒 | 重录 + 包含店铺外观 + 内部 + 业务证明 |
| Listing 不显示在 Maps | 通常验证后 24h 内出现，慢则 1 周 |
| 中文/英文用户搜不到 | 在 GBP 加 alternate name "Fukuraku Cattery" / "福楽猫舍" |
| 别人冒充我的店铺 | 后台 "Suggest an edit" → "Close or remove" → 申请 ownership |
